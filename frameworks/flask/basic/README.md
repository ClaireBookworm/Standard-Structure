## Minimal Application Template

```
├── main.py
├── static/
│   ├── script.js
│   ├── style.css
│   └── image.png
└── templates/
    └── index.html
```

### `main.py`

`main.py` is where you write your python code!

### `static/`

`static` is where all your static web assets go (css, js, images etc.).

### `templates/`

`templates` holds all the html files for the project.
